# Task: 实现 #2028 AF_PACKET send_raw_packet

## 背景
DragonOS AF_PACKET socket 当前无法发送原始以太网帧。`IfaceCommon::send_raw_packet()` 返回 ENOSYS。

## 架构关键发现
DragonOS 使用 smoltcp 0.12（fork），该版本**不让 Interface 持有 device**——`poll()` 接收 `&mut Device` 作为参数。device 在各驱动（VirtioInterface/VethInterface 等）自己手里。

当前调用链：
```
PacketSocket::try_send(buf, dest)
  → SOCK_RAW: 直接使用 buf 作为帧
  → SOCK_DGRAM: 构造以太网头 (dest_mac + src_mac + protocol + payload)
  → send_raw_frame(iface, frame)
    → iface.common().send_raw_packet(frame)  ← ENOSYS!
```

## 设计方案
在 `Iface` trait 新增 `raw_transmit` 方法（数据结构优先：解决放错位置的问题）。

### 需要修改的文件

1. **`kernel/src/driver/net/mod.rs`**：
   - 在 `Iface` trait 新增方法：
     ```rust
     /// 发送原始以太网帧（AF_PACKET 用）
     fn raw_transmit(&self, _frame: &[u8]) -> Result<(), SystemError> {
         Err(SystemError::ENOSYS)
     }
     ```
   - `IfaceCommon::send_raw_packet` 可以保留（标记 deprecated）或删除

2. **`kernel/src/driver/net/virtio_net.rs`**：
   - `VirtioInterface` 实现 `raw_transmit`：
     ```rust
     fn raw_transmit(&self, frame: &[u8]) -> Result<(), SystemError> {
         use smoltcp::phy::Device;
         let device = self.device_inner.force_get_mut();
         match device.transmit(smoltcp::time::Instant::now().into()) {
             Some(tx_token) => {
                 tx_token.consume(frame.len(), |buf| {
                     buf.copy_from_slice(frame);
                 });
                 Ok(())
             }
             None => Err(SystemError::ENOBUFS),
         }
     }
     ```
   - 参考已有的 `phy::Device` impl（L621-668）和 `phy::TxToken` impl（L670-683）

3. **`kernel/src/driver/net/veth.rs`**：
   - VethInterface 实现 `raw_transmit`，用同样的 `force_get_mut().transmit()` 模式
   - 参考 veth 的 `phy::Device` impl 中的 transmit 方法

4. **`kernel/src/driver/net/e1000e/e1000e_driver.rs`**：
   - E1000eInterface 实现 `raw_transmit`，同样模式

5. **`kernel/src/driver/net/loopback.rs`**：
   - LoopbackInterface 实现 `raw_transmit`（如果适用），或保持默认 ENOSYS

6. **`kernel/src/net/socket/packet/mod.rs`**：
   - `send_raw_frame` 改为调用 `iface.raw_transmit(frame)` 而非 `iface.common().send_raw_packet(frame)`

### 关键注意事项
- `force_get_mut()` 是 unsafe interior mutability，与 poll() 中已有的使用模式一致（L777）
- `smoltcp::time::Instant::now().into()` 将系统时间转为 smoltcp Instant
- TxToken::consume 返回值 R：对于 raw transmit，R = () 即可
- 如果 transmit() 返回 None（无可用 TX buffer），返回 ENOBUFS
- 需要在各驱动的 Iface impl 块中添加 raw_transmit 方法
